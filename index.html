Curriculum

各フェーズで学ぶこと

PHASE 1 : 5月・6月

AI画像生成
見えないものを見る。ビジョンを言葉にし、形にする

使用ツール：Canva / Seedream / ChatGPT
第1回（5月）: わたしという世界の色
課題設定: 「自分とは何か」を考え、世界観・色・イメージを言語化する。
実践: 自分の顔写真を加工・デザイン化してSNSアイコンに。ブランドロゴや名刺データをAIで制作し、自己紹介プレゼン資料を作ります。
第2回（6月）: ビジョンの具現化
課題設定: オラクルカード（図柄のみ）20枚分のイメージ原案を言葉にします。
実践: オラクルカード20枚の画像を生成し、デザインとして仕上げます。（※デジタル画像として完成）
PHASE 2 : 7月・8月

AI実践活用
書くことは自分を表現すること。「うまく書かねば」の呪縛を解く

使用ツール：Claude / ChatGPT / NotebookLM
第3回（7月）: 言葉を磨く・展開する
課題設定: オラクルカード20枚のメッセージを書く。日常や仕事でのAI活用シーンを見つける。
実践: AIを使ってカードの文言を磨き上げます。文章を、動画・インフォグラフィック・音声へと展開します。（NotebookLM使用）
第4回（8月）: 発信と自分の言葉
課題設定: ブログを立ち上げ（Ameblo / Substack等）、自らの語り口と発信テーマを決める。
実践: AIにオラクルカードのリーディング内容を出力させ、それを自分の解釈として語る練習をします。ブログ記事をAIで書き、その場で公開します。
PHASE 3 : 9月・10月

AI音楽制作
世界観を「音＝波動」で表現する。半年間の学びの統合

使用ツール：Claude / ChatGPT / Suno / Canva
第5回（9月）: 音で世界観を作る
課題設定: 画像・文章・カードから見えてきた「自分の世界観」を改めて言葉にする。
実践: Sunoを使って自分の世界観に合う音楽をひたすらクリエイト。ClaudeやChatGPTで歌詞をブラッシュアップし、様々なジャンルの音楽を生成する
第6回（10月）: 音 x 動画 で世界観を深める
課題設定: 楽曲とMV動画の最終仕上げを行い、6か月の全作品を振り返る。
実践: 楽曲・MV動画を発表。カード・ブログ・音楽など全作品を「自己紹介」にまとめる

Requirements

  • OS：Windows 10 または 11（64ビット）
  • CPU：Intel Core i5 / AMD Ryzen 5 以上
  • メモリ（RAM）：8GB以上（複数タブを快適に使うなら16GB推奨）

  • インターネット：安定した常時接続 10Mbps以上
  • 音声出力：イヤフォン・スピーカー（Suno・NotebookLM音声に必要）

Mac（推奨スペック）

  • OS：macOS 12 Monterey 以降
  • CPU：Apple Silicon（M1以降）または Intel Core i5以上
  • メモリ（RAM）：8GB以上（16GB推奨）
  • ストレージ：空き容量 10GB以上
  • ブラウザ：Google Chrome 最新版（推奨）/ Safari 最新版
  • インターネット：安定した常時接続 10Mbps以上
  • 音声出力：イヤフォン・スピーカー

【注意点】

  • Sunoは画面幅768px未満の端末（スマホ等）は非対応

全6か月の日程は最初から決まっていますか？
PHASE 1（5・6月）の日程は確定しています。PHASE 2・3は、ご参加の皆様の進行に合わせ、前フェーズ修了の1か月前までにご案内します。
50代でついていけますか？
この講座はまさに50代の方のために設計しています。「できない」が「できる」に変わる体験を、ゆっくり丁寧に積み重ねていきますのでご安心ください。
